PALOMA Test API Server

My app API test platform is described below. Some of the APIs require a request body to be sent (like /user/login and /user/register), while others require an Authorization authToken (received at login). To view the required JSON format, click on a command below and look at the example request body. To try out an API, modify the request as needed, and press Send.

HTTP Endpoints

[POST]
/user
 Register a user If successful, an authorization authToken is returned. You may use the authToken with future requests that require authorization. No authorization authToken is required to call this endpoint.
[POST]
/session
